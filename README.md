# prototype-mock

A single-file HTML prototype: a clickable iPhone/iPad device frame centered on a black stage, with step navigation, a menu popover, an EN/DE language switch, and a restart control.

## Usage

Open `prototype-mock.html` in a browser. No build step or dependencies.

## Files

- `prototype-mock.html` — the prototype (HTML, CSS, and JS all inline).
- `CLAUDE.md` — architecture notes for working on this file with Claude Code.
